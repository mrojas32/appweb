## Sobre el proyecto

El objetivo de este proyecto es desarrollar un sistema de reserva de canchas para nuestra universidad, el cual deberá facilitar la gestión y reserva de espacios deportivos por parte de estudiantes, personal docente y administrativo. Este sistema permitirá optimizar el uso de las instalaciones deportivas, garantizar una distribución equitativa del tiempo de uso y mejorar la experiencia de los usuarios al proporcionar un proceso de reserva sencillo y eficiente. 


En este proyecto deberán analizar los requisitos funcionales, diseñar una solución que los cumpla y desarrollar un prototipo del software que será evaluado por un comité de la universidad.
