Get Wireshark

\n

\n The current stable release of Wireshark is 1.8.3.\n It supersedes all previous releases, including\n all releases of Ethereal.\n You can also download the latest development release (1.8.0rc2)\n and documentation.\n

\n
\n \n \n
\n

\n Having problems? Explore our download area\n or look in our third party package list below.\n

\n

Installation Notes

\n

\n For a complete list of system requirements and supported platforms,\n please consult the User's Guide.\n

\n

\n Information about each release can be found in the release notes.\n

\n

\n Each Windows package comes with the latest stable release of\n WinPcap, which is required for live\n packet capture. If needed you can install the latest development release\n from the\n WinPcap download page.\n

\n

Live on the Bleeding Edge

\n

\n You can download source code packages and Windows installers which are\n automatically created each time code is checked into the source code repository. These\n packages are available in the automated\n build section of our download area.\n

\n \n

Stay Current

\n

\n You can stay informed about new Wireshark releases by subscribing to\n the wireshark-announce mailing list. We\n also provide a PAD file to\n make automated checking easier.\n
