Version 4.1.2 Installer for Windows

\n

Driver +DLLs

\n

Supported platforms:

\n
    \n
  • Windows NT4/2000
    • \n
  • Windows XP/2003/Vista/2008/Win7/2008R2 (x86 and x64)
    • \n
\n

MD5 Checksum: 929b7d846b635959201e30b57190284a
\n SHA1 Checksum: 5bbdce5c2ad5423ba023b1272301a7fb49279b16

\n

This executable file installs WinPcap on your machine.

\n
\n
\n
\n
\n

Instructions

\n
    \n
  1. Download and run the executable
    2. \n
  2. Follow the instructions on the screen. The installation applet will automatically detect the operating system and install the correct drivers
    3. \n
  3. The WinPcap-based applications are now ready to work
    4. \n
  4. To remove WinPcap from the system, go to the Control Panel, click on "Add/Remove programs" and then select "WinPcap"
    5. \n

Download WinPcap for Windows CE
\n
\n

Download the source code of a Windows CE version of WinPcap.

\n

Notes

\n
    \n
  • This is an experimental version, we don't guarantee it will work correctly in any situation.
    • \n
  • At the moment only source code is provided. You will have to compile it on your own.
    • \n
  • We didn't develop this code, so we don't provide any support for it.
    • \n
  • This version might not be aligned with the latest version of WinPcap.
    • \n
\n

The CE version was tested successfully (not by us) on Compaq Pocket PC and HP Jornada.

\n

Known Limitations:

\n
    \n
  1. The driver can be used to open only one adapter that is present in the system. not more than one adapter can be opened at any given point of time. Also, only one instance of the adapter can be opened.
    2. \n
  2. Statistics mode is not present.
    3. \n
  3. Only one request at a time is supported, requests are not queued.
    4. \n
  4. It works on Windows CE 3.0. may work on CE 2.x also but it was never tested.
    5. \n
